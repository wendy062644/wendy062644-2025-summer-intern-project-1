# 動機

## 我在解決什麼問題？
- 將分散的相片、軌跡、標註整合到單一互動式地圖中  
- 一鍵匯出可分享/可部署的成果（HTML / ZIP / KML / KMZ）

## 為什麼需要可視化/互動式操作？
- 地圖情境化：照片、路徑與地標一次看懂  
- 互動式檢索與快顯：比純相簿/純地圖更有效率

## 為什麼與 Jupyter Book 整合？
- **同站完成教學 + 展示 + 發佈**：步驟、原理、成果都放一起  
- **版本控管與自動部署**：GitHub Pages/CI 讓內容可追溯、可更新  
- **可重現性**：以文件帶流程，讓專題成果容易重現與擴充
